Operational Excellence

Design Principles

  • Operations as code — script everything
  • Small, reversible changes
  • Anticipate failure — pre-mortems & game days
  • Learn from every event

Observability Stack

  • CloudWatch — metrics, logs, alarms
  • X-Ray — distributed tracing
  • CloudTrail — API audit log

Is it healthy? Where's the latency? Who changed it?

Runbooks & IaC

  • Runbook = known procedure, known situation
  • Playbook = investigative steps for the unknown
  • CloudFormation / CDK — repeatable, reviewed, drift-detectable
  • Manual console changes are operational debt
