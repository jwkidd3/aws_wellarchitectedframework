Pillar 4

Performance Efficiency

Pick the Right Tool

  • Compute: EC2 / Containers / Lambda
  • Storage: S3, EBS, EFS — match access pattern
  • Database: RDS, Aurora, DynamoDB, ElastiCache
  • Workload shape drives the choice — not familiarity

Measure, Don't Guess

  • Load test before launch and after big changes
  • CloudWatch + X-Ray + Performance Insights
  • Target p95 / p99, not averages

Trade-Offs

  • Caching, read replicas, CDN
  • Each adds complexity — don't add until you've measured the need
